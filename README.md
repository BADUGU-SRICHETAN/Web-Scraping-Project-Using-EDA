# Shopsy Web Scraping Project

## Project Overview
This project focuses on extracting product data from Shopsy.com using Python web scraping techniques. The website's HTML content is parsed using BeautifulSoup, while Regular Expressions (Regex) are used to identify and extract specific patterns from the data.

The scraped data is cleaned, transformed into a structured format using Pandas, and prepared for further analysis.

## Objectives
- Scrape product information from Shopsy.com.
- Extract relevant details using BeautifulSoup and Regex.
- Clean and organize the collected data.
- Store the extracted data in a structured format for analysis.

## Technologies Used
- Python
- Jupyter Notebook
- BeautifulSoup
- Regular Expressions (Regex)
- Requests
- Pandas

## Project Workflow
1. Send HTTP requests to retrieve webpage content.
2. Parse HTML using BeautifulSoup.
3. Use Regex to extract specific product information.
4. Clean and preprocess the scraped data.
5. Store the data in a structured dataset.

- Data Cleaning
- Data Transformation
- Python Programming
